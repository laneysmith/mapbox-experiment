# mapbox-experiment

## Starting the app:

1. Create a Mapbox account and obtain an API access token [here](https://account.mapbox.com/)
1. Copy the `.env.example` file to `.env` and update with your access token:
   ```
   cp .env.example .env
   ```
1. Install dependencies and start the app:
   ```
   yarn install
   yarn start
   ```
